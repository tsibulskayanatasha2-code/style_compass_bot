# Style Compass Bot 🌸

Telegram-бот для подбора стильных образов под погоду и внешность.

## 📌 Возможности
- Анкета с 8 вопросами
- Подбор образа под погоду
- Сохранение в базе данных

## 🛠 Технологии
- Python + Aiogram
- PostgreSQL
- OpenWeatherMap API

## 👨‍💻 Команда
- Наташа — Тимлид, бэкенд
- Подруга 1 — Бот, анкета
- Подруга 2 — Контент, образы
